# Simple Banking System (Tycoon)
A simple banking API built with Go `v1.22`.

## Setup

```
make docker/build
make docker/up
```

## Migration

```
make db/migrate/create
make db/migrate/up
make db/migrate/down
```

## Testing

```
make tests
```
